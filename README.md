# Online Learning Management System  Developed using Django and SQLite. 
 ## Features  
 - User authentication  
 - Course management  
 - Assignment submission 
 - Admin dashboard  
 - Teacher dashboard  
 - Student dashboard 

 ## Live Website

The Online Learning Management System is deployed and accessible online.

Access the live system here:
https://online-lms-django-2.onrender.com

Users can register, log in as students or teachers, enroll in courses, and interact with the learning platform.
